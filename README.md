# İşlemTamam - LGS Hazırlık Platformu

Bu proje, 10 kişilik uzman öğretmen kadrosuyla LGS (Liselere Giriş Sınavı) hazırlık kursları sunan online eğitim platformunun web sitesidir.

## Platform Özellikleri

- **LGS Uzman Öğretmenler**: Her branşta 10+ yıl deneyimli öğretmen ekibi
- **Canlı LGS Dersleri**: Haftada 12+ saat interaktif LGS dersler
- **Soru Çözüm Sistemi**: 7/24 canlı soru çözümü ve feedback
- **LGS Deneme Sınavları**: Aylık deneme sınavları ve başarı analizi
- **Kişiye Özel Koçluk**: Zayıf branşlarda özel ilgi ve destek
- **Dijital Kütüphane**: Binlerce çıkmış LGS sorusu ve konu anlatımları

## Teknik Kurulum

**Gereksinimler:**  Node.js

1. Bağımlılıkları yükleyin:
   ```
   npm install
   ```
2. `.env.local` dosyasında gerekli ayarları yapılandırın:
   ```
   GEMINI_API_KEY=your_gemini_api_key
   ```
3. Uygulamayı başlatın:
   ```
   npm run dev
   ```

## Proje Yapısı

- `/components` - React bileşenleri (Hero, Features, Packages, vb.)
- `/index.html` - Ana HTML dosyası
- `/App.tsx` - Uygulamanın ana bileşeni
- `/metadata.json` - Proje metadata bilgileri
